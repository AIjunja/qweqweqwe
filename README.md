# Dark Velocity

Google Flow로 생성한 Tesla Roadster 콘셉트 영상을 사용하는 시네마틱 랜딩 페이지입니다.

## Live site

- GitHub Pages: https://aijunja.github.io/qweqweqwe/

## Features

- 풀스크린 Roadster 영상 히어로
- GSAP/ScrollTrigger 기반 스크롤 모션
- 성능 수치 카운트업
- SVG 공기 흐름 애니메이션
- 커스텀 커서와 마그네틱 인터랙션
- 데스크톱·모바일 반응형
- `prefers-reduced-motion` 대응
