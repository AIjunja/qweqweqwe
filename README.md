# Codex VIBE 코딩 실습 자료

캡처 기반으로 넘겨 보는 HTML 강의 자료입니다.

## 구성

- `index.html`: 슬라이드형 HTML 뷰어
- `styles.css`: Gmarket Sans 기반 스타일
- `script.js`: 목차, 개념 정의, 이미지 링크 매칭
- `assets/screenshots`: 공식 문서/서비스 캡처
- `assets/projects`: 프로젝트 이미지
- `assets/qr`: YouTube, Instagram, Threads, 수업자료 QR
- `assets/needed`: 로그인된 ChatGPT 앱에서 교체 캡처가 필요한 화면

## 사용법

브라우저에서 `index.html`을 열면 됩니다.

- 오른쪽 화살표 / Space: 다음 슬라이드
- 왼쪽 화살표: 이전 슬라이드
- 이미지 클릭: 원본 문서, GitHub, 서비스 페이지로 이동
